## Project Overview

The MATLAB GUI was designed to assist in the analysis of overhead transmission
line geometry by allowing engineers to study the impact of different electrical
and mechanical parameters on line performance.

The tool conceptually supported:
- Input of transmission line configuration parameters (conductor spacing,
  geometry arrangement, and physical properties)
- Evaluation of electrical characteristics influenced by line geometry
- Analysis of how geometric variations affect transmission efficiency and losses
- Interactive exploration of parameter changes through a graphical user interface

The project emphasized engineering insight and visualization rather than
automation or large-scale simulation.
